# Assessments

Course assessment structure and grading rubric.

## Weekly Quizzes (20%)

Multiple choice and short answer questions covering lecture material.

## Lab Assignments (30%)

Hands-on coding assignments:
- ROS 2 package development
- Robot simulation
- Sensor integration

## Midterm Project (20%)

Build a multi-node robotic system with:
- Sensor fusion
- Motion planning
- Control

## Final Capstone (30%)

**Autonomous Humanoid Project:**
- Voice command recognition
- Path planning and navigation
- Object manipulation
- Full system integration

### Grading Rubric
- Code Quality: 30%
- Functionality: 40%
- Documentation: 15%
- Innovation: 15%
